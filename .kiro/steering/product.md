# Product Overview

## Modular Financial Reporting System v0.2.0

A Python-based automated financial reporting system designed for Dutch GAAP (Generally Accepted Accounting Principles) compliance. The system processes financial data from XAF (XML Audit File) format and generates professional annual reports using Quarto.

### Core Capabilities

- **XAF Processing**: Extracts and processes trial balance, transactions, and chart of accounts from XML audit files
- **Financial Statement Generation**: Automatically creates income statements (winst-en-verliesrekening) and balance sheets (balans) following Dutch GAAP structure
- **Modular Report Architecture**: Separates data processing logic from presentation templates for maintainability
- **Multi-format Output**: Renders reports to HTML, PDF, and DOCX formats
- **Business Rules Validation**: Validates financial data against Dutch GAAP requirements
- **Sample Data Mode**: Includes Telco B.V. sample data for testing and demonstration

### Target Use Case

Designed for Dutch companies (specifically Telco B.V.) to automate annual financial report generation from accounting system exports, ensuring compliance with Dutch reporting standards while maintaining flexibility for customization.
